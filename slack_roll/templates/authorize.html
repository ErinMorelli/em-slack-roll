{{ project.name_full }} needs a Slack API Token with posting privileges to work:

How to Get a Token

  1. While logged into Slack, visit this link.

  2. Click on the green Create Token button next to the team you would like to authorize EM Slack Roll to post in.

  3. Finally, enter the generated token into the form field above and click Authorize Token to get rolling!
